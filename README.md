# Permaculture 4 Squares (P4S)

**Version 0.9.5.26** — Open-source permaculture grid simulator built with **Godot 4.x**. Open **this folder** in Godot (the one that contains `scenes/`, `scripts/`, and `data/`) via **`project.godot`**. Do not open nested folders like `p-4s-8.5.26/` — those are old stubs without game files.

## Documentation for developers

- **[Codebase guide](docs/CODEBASE_GUIDE.md)** — How scenes, autoloads, the map script, HUD, and `data/` fit together. Read this before changing systems you did not write.
- **[Architecture](docs/ARCHITECTURE.txt)** — Scene flow, autoloads, and persistence boundaries.
- **[Open Core](obsidian/10_Open_Core_Architecture.md)** — Asset split and public vs premium repo rules.

File-level **`##` headers** in scripts point to the codebase guide when useful.
